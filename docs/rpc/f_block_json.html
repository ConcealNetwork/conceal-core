Get Block Details

Retrieve details of a block using JSON-RPC.

Request Body schema: application/json
jsonrpc
required
string
Value: "2.0"
method
required
string
Value: "f_block_json"
required
object (GetBlockParams)
id
required
string

Responses

http://localhost:16000/json_rpc

Testnet local node

http://localhost:16600/json_rpc
